# DECISIONS — ws-cursor

Bu klasör Cursor IDE ile kullanım için talimatlar workspace'idir.

## Çalışma modları
- **API'siz (yerel):** `APPLY=true npm run talimatlar-ai` veya `npm run talimatlar-watch` — sadece script içi kalıplar ve heuristic'ler; Cursor API/agent gerekmez.
- **Cursor Agent (API):** `npm run talimatlar-agent` — Cursor CLI kurulu ve `agent login` yapılmış olmalı; doğal dil talimatları AI uygular.

Bu dosya TALIMATLAR.md maddeleriyle güncellenir.
