# demo-run — Tek satırlık senaryo

Demo run senaryosu (örnek):
1. spec: `project_spec.md` ve `task_backlog.json` oluşturulur.
2. consensus: ekip üyeleri PR üzerinde onay verir.
3. ide_run: `npm run test` veya `node scripts/run-demo.js` ile IDE/CI içinde çalıştırılır.
4. validation: smoke test'ler ve basit entegrasyon testleri çalıştırılır.
5. deploy (opsiyonel): insan onayı sonrası production deploy yapılır.

Not: Bu belge örnektir; detaylar TASKS.md ve DECISIONS.md içinde tanımlıdır.
