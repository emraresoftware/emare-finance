# TASKS — Özet görev listesi

Bu dosya örnek run/demo görevlerini ve sorumluluklarını özetler.

## Demo run görevleri
- Hazırla: project_spec.md ve task_backlog.json oluştur
- Consensus: team onayı alın
- ide_run: IDE içinde otomatik çalıştırma
- validation: testler çalışsın, smoke test geçsin
- deploy: (opsiyonel) production deploy — human approval gerekli
