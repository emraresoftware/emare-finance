# ws-2-model-broker çıktı klasörü

**Model-broker sadece bu klasöre (`proje/`) yazar.** Consensus giriş/çıkış, provider cevapları vb. buraya konur. Karışıklık olmaması için diğer agent’ların klasörüne yazılmaz.

- Proje bazlı alt klasör istersen: `proje/ornek-proje-api/` gibi kullanılabilir.
- DevM kuralı: **D-005** — her agent kendi workspace’indeki `proje/` altına yazar.
