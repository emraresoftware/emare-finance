# ws-3-ide-runner çıktı klasörü

**IDE-runner sadece bu klasöre (`proje/`) yazar.** Talimat dosyaları, loglar, sonuçlar vb. buraya konur. Karışıklık olmaması için diğer agent’ların klasörüne yazılmaz.

- Proje bazlı alt klasör istersen: `proje/ornek-proje-api/` gibi kullanılabilir.
- DevM kuralı: **D-005** — her agent kendi workspace’indeki `proje/` altına yazar.
